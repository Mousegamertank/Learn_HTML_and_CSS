	<h1></h1> {Titulo}
	
	<img src="" alt=""> {imagens}
	
	<input type=""> {caixas de interação}
	
	<a href="" title="" target=""> texto </a> {tag para links}
	
	<div></div> {usada para criar blocos de conteudo}
	
	<p></p> {tag para paragrafo}

	<!DOCTYPE html> {usado apenas para indicar ao navegador que é HTML 5}

	<html lang=""></html> {usada para indicar qual codigo html vai ser mostrado, o atributo lang define a lingua padão do site}

	<head></head> {usado como elemento dentro do html junto com o <body> que serve para configurações importantes para a pagina porém sem que o usuario veja}

	<body></body> {corpo do documento assim como o head, serve para a construção da página em si}

	<title> <title> {quando usado no head como tag, defina o nome da pagina}

	<meta charset="utf-8"> {meta são metadados que serão usados na página, a tag é utlizada no head ,o atributo charset define o tipo de teclado disponivel para digitação no caso UTF-8 aceita até 'ç'}

	<h1></h1> {texto de maior importancia}
	
	<h2></h2> {texto com a segunda maior importancia}

	{tags <h1> até <h6> são tags para dar importancia ao texto}

	<li></li> {item de uma lista}

	<ol></ol> {lista ordenada}

	<blockquote cite="aqui vem sua citação"> <p> Aqui o texto a ser citado dentro de um p </p></blockquote>

	<cite> Quem citou ou citação aqui </cite>

	<q cite="citação ou link da citação aqui"> o que foi falado na citação </q>

	<abbr title="A extensão aqui"> Texto abreviado aqui </abbr>

	<address> Endereço de quem fez o HTML </address>

	<dl> cria uma lsita que pode ter descrição </dl>

	<dt> termo da dl que serve como o termo principal </dt>

	<dd> termo da dl que serve com desrcição da dt </dd>

	<pre> Bloco grande de conteudo de codigo </pre>

	<code> Bloco pequeno de conteudo de codigo </code>

	<span> Agrupar Linhas </span>

	<table> Criar tabelas </table>

	<tr> Linha dentro da Table </tr>

	<th rowspan="núm inteiro para controle de linhas" colspan="núm inteiro para controle de colunas", scope="col, row"> Cabeçalho dentro da linha </th>

	<td> Descrição </td>

	<thead> Usado para indicar os titulos da tabela </thead>

	<tbody> Usado para indicar o corpo da tabela </tbody>

	<tfoot> Usado como area para marcar resultados </tfoot>

	<captions> Descrever o que é a tabela </captions>

	<colgroup> possui a tag col dentro e nela voce consegue fazer as stilizações necessarias para sua coluna </colgroup>

	<col style="CSS"> esta tag vai pegar coluna por coluna da sua tabela </col>
